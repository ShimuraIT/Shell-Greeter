# Good afternoon, user.
is just about all it should say, ever.
if you want to rng out a motivational quotation, go ahead.
if you want emojis, go nuts.

### Highly customizable
because it's 40 lines of shell script.
it greets you with your first name, because GECOS has no spot for an honorific; e.g. "Good afternoon, Mr. Shimura."
it would be a great deal more complex, but you could ask and store that info.

### Installation
run it with your shell rc script i.e. `~/.bashrc`
